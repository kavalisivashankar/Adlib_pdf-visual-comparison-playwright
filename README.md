# PDF Visual Comparison Framework (Playwright)

## Overview
This project automates PDF comparison using Playwright by:
- Converting PDF pages to images
- Performing pixel-by-pixel visual comparison
- Generating diff images
- Producing an HTML report with PASS/FAIL status

## Setup

1. Install dependencies
   npm install

2. Install Playwright browsers
   npx playwright install

3. Run test
   npx playwright test

## Output
Results will be generated in:
results/report.html
